E-Commerce Website

A simple and responsive E-Commerce Website that allows users to browse products, place orders, and manage their accounts. It also includes an admin panel for managing products, categories, customer orders, and users efficiently.

Features :
User Registration & Login
Product Categories
Product Search
Checkout System
Order Management
Admin Dashboard
Product Management
Category Management
Yajra DataTables Integration
Responsive Design
Project Overview

This project provides a complete online shopping system where users can explore products and place orders easily. The admin panel helps in managing all website operations including products, categories, and customer orders. The system is designed to be fast, user-friendly, and efficient for both users and administrators.
